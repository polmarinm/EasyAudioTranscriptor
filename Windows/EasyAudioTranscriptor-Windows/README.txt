Transcriptor de Audios

Este programa te permite convertir audios en texto de manera sencilla.

REQUISITOS - ¿Qué necesitas?
    1. Una computadora con Windows 10 o superior.
    2. Conexión a Internet (solo para la instalación).
    3. Permisos de administrador (solo para la instalación).

PASO 0 - Instalación (Solo la primera vez)
    1. Haz clic derecho en "00_instalar_dependencias.bat" y selecciona "Ejecutar como Administrador".
    2. Espera a que se instalen las herramientas necesarias (puede tardar unos minutos).
    3. Cuando termine, verás un mensaje de éxito. ¡Listo! Ahora puedes usar el programa sin conexión a Internet.

PASO 1 - Limpiar las carpetas (Opcional)
    1. Si deseas limpiar todas las grabaciones anteriores, ejecuta "01_limpiar.bat".
    2. Confirma la eliminación cuando se te pregunte. 
    3. Se eliminará todo el contenido de las carpetas "audios/" y "transcripciones/".

PASO 2 - Transcribir audios
    1. Pon los archivos de audio en la carpeta "audios/".
    2. Abre "02_transcriptor.bat" con doble clic.
    3. Espera a que termine el proceso.
    4. Abre la carpeta "transcripciones/" para ver los archivos de texto generados.

¡Eso es todo! Sigue los pasos y convierte tus audios en texto sin complicaciones.
